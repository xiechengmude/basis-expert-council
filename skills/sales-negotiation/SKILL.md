# 商务谈判与销售成单 Skill

## 何时使用

当需要：
- 制定家长咨询到签约的完整销售流程
- 设计报价方案和课包结构
- 提升客单价和续费率
- B 端（学校/机构）商务谈判
- 合同条款设计和风险控制

## C 端（家长）销售全流程

### 阶段 1：引流获客

**线上渠道：**
| 渠道 | 策略 | 预期成本 |
|------|------|---------|
| 小红书 | "BASIS 家长必看" 系列笔记 | 内容成本 |
| 微信公众号 | "BASIS 学习攻略" 文章 | 内容成本 |
| 朋友圈广告 | 定向 BASIS 校区周边 3km | 50-200 元/线索 |
| 百度/搜狗 SEM | "贝赛思辅导" 关键词 | 100-300 元/线索 |
| 短视频 | "BASIS 老师教你..." 系列 | 内容成本 |

**线下渠道：**
| 渠道 | 策略 | 预期成本 |
|------|------|---------|
| 校门口 | 放学时段发传单+扫码 | 人工+物料 |
| 家长讲座 | 每月社区/酒店讲座 | 场地 2000-5000 |
| 异业合作 | 英语机构/留学中介互推 | 分成 |
| 老客转介绍 | 推荐成功送 2 课时 | 课时成本 |

### 阶段 2：首次咨询（黄金 30 分钟）

**咨询流程：**

```
前 5 分钟：建立关系
├── 自我介绍 + 机构介绍（简洁）
├── 询问孩子基本情况
└── 表达对孩子的关心（不要急于推销）

中间 15 分钟：需求诊断
├── SPIN 提问法挖掘痛点
├── 记录关键信息（年级、学科、成绩）
├── 展示你对 BASIS 体系的专业理解
└── 让家长感觉"终于遇到懂的人了"

后 10 分钟：方案呈现
├── 基于诊断结果推荐方案
├── 展示类似案例的成功故事
├── 报价 + 限时优惠（如果适用）
└── 明确下一步（预约测评/体验课）
```

**关键话术：**
- **不要说**："我们的课程很好" → **要说**："您孩子的情况，我们之前有一个类似的学生，3 个月从 C 提到了 A-"
- **不要说**："别的机构不行" → **要说**："我们专注 BASIS 体系，教研团队每学期都会更新课程，确保和学校进度完全一致"
- **不要说**："赶紧报名" → **要说**："建议先做一次免费评估，有了准确数据再做决定更科学"

### 阶段 3：学科测评（转化关键）

这是最重要的转化环节——**用专业度征服家长**。

**测评内容：**
1. 学术英语水平测试（20 分钟）
2. 核心学科能力测试（30 分钟）
3. 学习习惯问卷（10 分钟）

**测评报告呈现（面对面）：**
```
"根据测评结果，您孩子的情况是这样的——
数学能力对标 BASIS G7 水平是 [X]，主要差距在 Word Problems 和 Reasoning...
英语学术水平在 Level [X]，具体来说...
我们建议的方案是..."
```

**核心策略**：让家长看到具体数据，产生"问题比想象的严重"的认知。

### 阶段 4：方案报价

**课包设计原则：**

| 课包 | 课时 | 单价 | 总价 | 定位 |
|------|------|------|------|------|
| **体验包** | 4 课时 | 1000 元/时 | 4,000 | 低门槛试水 |
| **短期包** | 20 课时 | 900 元/时 | 18,000 | 一个月密集 |
| **学期包** | 60 课时 | 800 元/时 | 48,000 | 一学期同步 |
| **年度包** | 120 课时 | 700 元/时 | 84,000 | 最优性价比 |
| **全托管 VIP** | 不限课时 | 按月 | 15-25 万/年 | 最高端 |

**报价技巧：**
1. **先报高再报低**：先说年度包的单价优势，再说短期包
2. **对比锚定**：和学费对比（"只是学费的 X%"）
3. **分期支付**：大额课包可分 2-3 期付款
4. **限时优惠**：首期学员 95 折（制造紧迫感）

### 阶段 5：成交 & 签约

**签约文件清单：**
- [ ] 辅导服务协议（含退费条款）
- [ ] 学生学习档案（记录基线数据）
- [ ] 辅导计划书（具体到每周内容）
- [ ] 教师介绍（建立信任）

**退费政策（建议）：**
- 首 4 课时内不满意，全额退款（消除顾虑）
- 4 课时后，按剩余课时 80% 退费
- 全托管方案按月结算

## B 端（学校/机构）谈判

### 目标合作对象

| 对象 | 合作内容 | 谈判策略 |
|------|---------|---------|
| **BASIS 离职教师** | 兼职授课/课程顾问 | 按课时分成 + 顾问费 |
| **其他国际学校** | 教师 EMI 培训 | 按人头收费，展示培训效果 |
| **留学中介** | 互推客户 | 推荐提成（10-15%） |
| **英语培训机构** | 联合课程 | 共享收入 |
| **教育地产/社区** | 场地 + 引流 | 降低场地成本 |

### B 端谈判要点

**与学校谈教师培训合作：**
```
价值主张：
"我们可以帮助贵校的中文教师提升英语学科教学能力（EMI），
降低对外籍教师的依赖。培训后教师可获得 EMI 教学认证。"

报价参考：
- 集中培训（5 天）：5-8 万/校
- 学期驻校培训：10-15 万/学期
- 年度合作：20-30 万/年
```

**与离职教师谈合作：**
```
方案 A：兼职授课
- 课时费：300-500 元/小时
- 不影响主业，灵活安排

方案 B：合伙人
- 基础薪资 + 课时费 + 利润分成
- 参与教研，共建课程体系

方案 C：顾问
- 月度顾问费 5000-10000
- 提供课程审核和教师培训指导
```

## 客单价提升策略

### 追加销售（Upsell）

| 时机 | 追加产品 | 话术 |
|------|---------|------|
| 签约时 | 学科 1→2 科 | "同时补两科效率更高，第二科 9 折" |
| 第一月后 | 短期→学期包 | "续费学期包单价更优惠" |
| 考试季 | AP 冲刺班 | "考前 6 周冲刺班，专攻 FRQ" |
| 学年末 | 暑期衔接班 | "暑期预修下学期内容，开学领先" |
| 升学阶段 | 升学规划 | "综合规划 AP 选课+大学申请" |

### 续费策略

- **学期末复盘会**：展示学生进步数据，自然引出续费
- **续费优惠**：老学员续费 95 折 + 赠送 2 课时
- **锁定长期**：年度包优惠 > 学期包 > 短期包
- **社群粘性**：日常社群运营维持关系

## KPI 指标体系

| 指标 | 目标值 | 计算方式 |
|------|--------|---------|
| 线索获取成本 | <200 元 | 营销费用 / 新线索数 |
| 咨询→测评转化率 | >60% | 预约测评数 / 总咨询数 |
| 测评→签约转化率 | >50% | 签约数 / 测评数 |
| 平均客单价 | >3 万 | 总收入 / 签约学生数 |
| 续费率 | >75% | 续费学生 / 到期学生 |
| 转介绍率 | >30% | 转介绍新生 / 在读学生 |
| 月均收入 | >30 万（起步期） | 当月总收入 |

## 合同模板要点

### 必须包含的条款

1. **服务内容**：具体学科、课时数、授课方式
2. **费用条款**：总价、支付方式、发票
3. **退费条款**：退费计算方式、退费时限
4. **上课安排**：时间、地点、调课规则
5. **教师安排**：指定教师 / 可更换条件
6. **保密条款**：学生信息保密
7. **争议解决**：协商 → 仲裁
8. **免责声明**：不承诺具体分数/排名提升

### 避免的法律风险
- **不要承诺"保分"/"保提升"** — 用"针对性辅导方案"代替
- **不要诋毁学校** — 只分析体系特点，不评价学校好坏
- **教师资质合规** — 确保教师有合法执教资格
- **课外辅导资质** — 确认当地的经营许可要求
