## Essay 写作核心技能

### 5-Paragraph Essay 基本结构

```
Introduction（引言）
├── Hook（开头吸引句）
├── Background Information（背景信息）
└── Thesis Statement（中心论点）——最关键！

Body Paragraph 1（主体段1）
├── Topic Sentence（主题句）
├── Evidence / Example（证据/例子）
├── Analysis / Explanation（分析/解释）
└── Transition（过渡句）

Body Paragraph 2（主体段2）
├── Topic Sentence
├── Evidence / Example
├── Analysis / Explanation
└── Transition

Body Paragraph 3（主体段3）
├── Topic Sentence
├── Evidence / Example
├── Analysis / Explanation
└── Transition

Conclusion（结论）
├── Restate Thesis（重述论点，换种说法）
├── Summary of Main Points（总结要点）
└── Final Thought / Call to Action（收尾思考）
```

### Thesis Statement 写作公式

**Argumentative Essay：**
> "Although [counterargument], [your position] because [reason 1], [reason 2], and [reason 3]."

**Literary Analysis：**
> "In [title], [author] uses [literary device 1] and [literary device 2] to [convey/reveal/illustrate] [theme/message]."

**DBQ (Document-Based Question)：**
> "[Historical claim] as evidenced by [document types/perspectives], which demonstrates [broader significance]."

### 文学分析核心技巧

**Close Reading 四步法：**
1. **Observe** — 标注文字特征（word choice, imagery, tone, syntax）
2. **Interpret** — 这些特征意味着什么？
3. **Connect** — 如何服务于整体 theme/argument？
4. **Evaluate** — 作者为什么做出这样的选择？

**常用 Literary Devices：**

| Device | 中文 | Definition | Example |
|--------|------|-----------|---------|
| Metaphor | 隐喻 | Comparison without like/as | "Life is a journey." |
| Simile | 明喻 | Comparison using like/as | "Her eyes were like stars." |
| Symbolism | 象征 | Object represents an idea | The green light in Gatsby |
| Irony | 反讽 | Contrast between expectation and reality | Dramatic, Verbal, Situational |
| Allusion | 典故引用 | Reference to another work | Biblical, Mythological |
| Imagery | 意象 | Sensory language | Visual, Auditory, Tactile |
| Tone | 语调 | Author's attitude | Sarcastic, Somber, Hopeful |
| Foreshadowing | 伏笔 | Hints at future events | Ominous weather before tragedy |

## 历史 DBQ / LEQ 写作指导

### DBQ (Document-Based Question) 结构

**时间分配（60 分钟）：**
- 阅读和分析文档：15 分钟
- 规划：5 分钟
- 写作：40 分钟

**评分标准（7 分制）：**
1. **Thesis/Claim（1 分）** — 明确论点，回应题目
2. **Contextualization（1 分）** — 历史背景（至少 2-3 句）
3. **Evidence from Documents（3 分）** — 至少引用 3-6 个文档
4. **Outside Evidence（1 分）** — 文档之外的历史知识
5. **Complex Understanding（1 分）** — 展示深入理解（因果、比较、连续性等）

### LEQ (Long Essay Question) 结构

**评分标准（6 分制）：**
1. Thesis（1 分）
2. Contextualization（1 分）
3. Evidence（2 分）
4. Analysis & Reasoning（2 分）

## 中国学生常见问题及应对

### 问题 1：阅读量跟不上
**应对：**
- 从短文开始，逐步增加长度
- 教授 skimming（略读）和 scanning（扫读）技巧
- 建立每日英文阅读习惯（30 分钟/天）
- 使用 annotation（批注）保持专注

### 问题 2：Essay 没有 argument
**应对：**
- 先练习写 Thesis Statement（每天一个话题）
- 使用 "because" 结构强制给出理由
- 教授 "They Say / I Say" 论证模式
- 区分 summary（总结）和 analysis（分析）

### 问题 3：课堂讨论不参与
**应对：**
- 准备 discussion sentence starters：
  - "I agree with [name] because..."
  - "Building on that point, I think..."
  - "I respectfully disagree because..."
  - "Can you clarify what you mean by...?"
- 课前预习讨论话题，准备 2-3 个观点

## Rhetorical Analysis 框架（AP English Language）

**SOAPSTone 分析法：**
- **S**peaker — 谁在说/写？
- **O**ccasion — 什么场合/背景？
- **A**udience — 受众是谁？
- **P**urpose — 目的是什么？
- **S**ubject — 主题是什么？
- **Tone** — 语调/态度是什么？

**常用修辞手法（Rhetorical Devices）：**
- Ethos（人格诉求）— 可信度和权威
- Pathos（情感诉求）— 情感共鸣
- Logos（逻辑诉求）— 事实和逻辑推理
- Anaphora（首语反复）— 重复句首词语
- Antithesis（对句/对比）— 对立概念并列
- Parallelism（排比）— 结构相似的句式
