# 国际教育同步辅导 — 可行性分析与商业模型方案

> 优先级：P0（正在执行）
> 产品：BASIS Expert Council

---

## 一、市场可行性分析

### 1.1 市场规模

| 指标 | 数据 |
|------|------|
| 中国国际学校数量 | ~1,400 所（2025） |
| 在读学生数 | ~60 万人 |
| 课外辅导渗透率 | 70-85%（国际学校家庭） |
| 人均年辅导支出 | 3-15 万元 |
| 可服务市场（SAM） | ~200-500 亿元/年 |
| AI 辅导可切入份额（SOM, 3 年） | 1-5 亿元 |

### 1.2 目标用户画像

- **核心用户**：贝赛思/包玉刚/世外/鼎石等头部国际学校 G5-G12 学生家长
- **年龄段**：家长 35-50 岁，学生 10-18 岁
- **家庭年收入**：50 万+
- **痛点排序**：
  1. 全英文教材跟不上进度（尤其理科）
  2. 找不到懂国际课程体系的好老师
  3. 现有辅导机构质量参差不齐、价格虚高
  4. 课后答疑无即时响应

### 1.3 竞争格局

| 竞争者类型 | 代表 | 优势 | 劣势 |
|-----------|------|------|------|
| 传统辅导机构 | 学而思国际/翰林学院 | 品牌、师资 | 贵（300-1500 元/时）、排课不灵活 |
| 1v1 私教 | 独立教师 | 个性化 | 供给极度有限、质量不稳定 |
| 通用 AI 工具 | ChatGPT/Kimi | 免费、响应快 | 不懂课程体系、无教学法设计 |
| 垂直 AI 辅导 | 几乎空白 | — | — |

**核心竞争优势**：
- 多 Agent 架构覆盖全学科，每个 Agent 内置学科教学法
- 深度理解 BASIS 等学校的课程节奏和考试模式
- Mem0 记忆系统实现跨会话个性化（记住学生薄弱点）
- 成本是真人教师的 1/50-1/100

### 1.4 可行性判定

| 维度 | 评分（1-5） | 说明 |
|------|------------|------|
| 技术可行性 | ⭐⭐⭐⭐⭐ | 已有 MVP，多 Agent + 记忆系统已跑通 |
| 市场需求 | ⭐⭐⭐⭐⭐ | 刚需高频，家长焦虑驱动 |
| 竞争壁垒 | ⭐⭐⭐⭐ | 课程知识库 + 用户数据飞轮 |
| 变现确定性 | ⭐⭐⭐⭐ | 家长付费意愿强，客单价高 |
| 监管风险 | ⭐⭐⭐⭐ | 国际教育不在双减范围内 |

**结论：高度可行，已在执行，应全力推进。**

## 二、执行指南

### 2.1 产品路线图

```
Phase 1: MVP 验证（当前 → 2026 Q1）
├── 完成 5 学科 Agent 调优
├── Mem0 记忆系统上线
├── 基础付费体系（免费试用 + 订阅）
├── 微信 H5 + 手机号登录
└── 目标：50 个付费用户

Phase 2: 增长期（2026 Q2-Q3）
├── 增加 IB/A-Level 课程覆盖
├── 作业拍照识别 + AI 批改
├── 家长端仪表盘（学习报告）
├── 推荐系统（薄弱知识点推送）
└── 目标：500 个付费用户

Phase 3: 规模化（2026 Q4-2027）
├── 与国际学校 B 端合作
├── 教师端工具（辅助备课/出题）
├── 社区 + 家长互助功能
├── 多语言支持（英文界面）
└── 目标：5,000+ 付费用户
```

### 2.2 获客策略

|------|------|---------|--------|

**核心获客钩子**：免费学科测评（已在开发）— 家长最关心「孩子到底什么水平」。

## 三、商业模型方案

### 3.1 收入模型

#### C 端订阅（核心）

| 套餐 | 月费 | 包含 | 目标转化率 |
|------|------|------|-----------|
| 免费体验 | 0 元 | 每日 5 次对话 + 1 次测评 | — |
| 基础版 | 49 元/月 | 每日 30 次对话、3 科 | 15% |
| 标准版 | 99 元/月 | 无限对话、全科、学习报告 | 50% |
| 高级版 | 199 元/月 | 标准版 + 作业批改 + 专属计划 | 25% |
| 家庭版 | 299 元/月 | 3 个孩子账号、家长仪表盘 | 10% |

**年付折扣**：8 折（提高 LTV、降低流失）

#### B 端合作

| 模式 | 收费 | 说明 |
|------|------|------|
| 学校 SaaS 授权 | 5-20 万/年/校 | 白标部署，学校品牌 |
| 辅导机构 OEM | 按学生数，50-100 元/学生/月 | 机构二次销售 |
| API 接口 | 按调用量计费 | 第三方教育平台集成 |

### 3.2 成本结构

| 成本项 | 月均（Phase 1） | 月均（Phase 3） | 说明 |
|--------|----------------|----------------|------|
| AI 模型调用 | 2,000 元 | 30,000 元 | 最大变动成本 |
| 云服务器 | 1,500 元 | 8,000 元 | Docker 部署 |
| 向量数据库 | 500 元 | 3,000 元 | Qdrant |
| 运营/内容 | 0 | 15,000 元 | 人力成本 |
| 获客 | 2,000 元 | 20,000 元 | 渠道投放 |
| **合计** | **~6,000 元** | **~76,000 元** |  |

### 3.3 单位经济模型

```
关键假设（Phase 2 目标状态）：
- ARPU = 99 元/月
- 平均留存月数 = 8 个月
- LTV = 99 × 8 = 792 元
- CAC = 50 元（社群获客为主）
- AI 成本/用户/月 = 15 元（对话 + 记忆检索）
- 毛利率 = (99 - 15) / 99 = 85%
- LTV/CAC = 792 / 50 = 15.8x ✅ 健康（>3x 即可）
```

### 3.4 财务预测

| 指标 | 2026 Q2 | 2026 Q4 | 2027 Q2 |
|------|---------|---------|---------|
| 付费用户数 | 100 | 500 | 2,000 |
| MRR | 9,900 | 49,500 | 198,000 |
| 月成本 | 15,000 | 40,000 | 100,000 |
| 月利润 | -5,100 | +9,500 | +98,000 |
| 累计现金流 | -30,000 | +20,000 | +400,000 |

**盈亏平衡点**：约 150 个付费用户（ARPU 99 元/月）。

### 3.5 风险与对策

| 风险 | 概率 | 影响 | 对策 |
|------|------|------|------|
| AI 回答质量不稳定 | 中 | 高 | 学科专家审核 + 用户反馈闭环 |
| 大厂入局（字节/好未来） | 中 | 高 | 深耕垂直（BASIS 等特定学校）形成壁垒 |
| 政策变化（AI 教育监管） | 低 | 高 | 定位为「学习工具」而非「培训机构」 |
| 获客成本上升 | 中 | 中 | 口碑传播 + B 端渠道降低对投放依赖 |
| 模型成本居高不下 | 低 | 中 | 用开源模型替代、缓存高频问答 |

## 四、总结

国际教育同步辅导是当前最确定的方向，已有产品基础。核心策略：

1. **Phase 1 聚焦 BASIS 单校验证**：先把一所学校打透
2. **用免费测评获客**：低成本获取精准用户
3. **99 元/月标准版为主力**：在「免费 AI」和「真人辅导」之间找到定价甜区
4. **150 用户即盈亏平衡**：目标明确、可快速验证
5. **数据飞轮是终极护城河**：用户使用越多，AI 越懂这所学校的课程体系
