# 标化考试自适应备考 AI — 可行性分析与商业模型方案

> 优先级：P2（3 个月内启动）
> 与教育主线高度协同，技术复用度高

---

## 一、市场可行性分析

### 1.1 市场规模

| 考试类型 | 中国年考生数 | 人均备考支出 | 市场规模 |
|---------|------------|-------------|---------|
| 托福 TOEFL | ~30 万 | 1-5 万 | ~60 亿 |
| 雅思 IELTS | ~40 万 | 1-5 万 | ~80 亿 |
| SAT/ACT | ~10 万 | 1-3 万 | ~15 亿 |
| GRE/GMAT | ~15 万 | 0.5-2 万 | ~15 亿 |
| AP 考试 | ~15 万 | 0.5-2 万 | ~15 亿 |
| **合计** | **~110 万** | — | **~185 亿** |

### 1.2 目标用户画像

- **核心用户**：准备出国的高中生和大学生
- **年龄段**：15-25 岁
- **特征**：时间紧迫、目标分数明确、愿意为提分付费
- **高度重叠**：与 P0（教育辅导）和 P1（留学规划）用户完全重叠

### 1.3 行业痛点

| 痛点 | 现状 | AI 解法 |
|------|------|--------|
| 刷题盲目 | 大量刷题但不知薄弱点 | 知识图谱 + 自适应推题 |
| 写作提升慢 | 没有即时批改反馈 | AI 实时评分 + 详细修改建议 |
| 口语练习难 | 缺少对练对象和反馈 | 语音对话 + 实时纠错评分 |
| 计划不科学 | 不知如何分配备考时间 | 智能备考计划 + 动态调整 |
| 真题资源散 | 各处找真题、答案不准 | 结构化题库 + 官方解析 |

### 1.4 竞争格局

| 竞争者 | 代表 | 优势 | 劣势 |
|--------|------|------|------|
| 传统培训 | 新东方/环球雅思 | 品牌、师资 | 贵（2-5 万/期）、效率低 |
| 在线班课 | 小站教育/考满分 | 价格适中 | 互动性差 |
| 题库 App | 雷哥/小猿搜题 | 免费、题量大 | 无个性化、无自适应 |
| 通用 AI | ChatGPT | 免费 | 不懂考试评分标准 |
| **自适应 AI 备考** | **极少** | — | — |

### 1.5 可行性判定

| 维度 | 评分 | 说明 |
|------|------|------|
| 技术可行性 | ⭐⭐⭐⭐ | 需要构建题库和知识图谱 |
| 市场需求 | ⭐⭐⭐⭐⭐ | 刚需，目标分数驱动 |
| 竞争壁垒 | ⭐⭐⭐⭐ | 自适应算法 + 题库数据 |
| 变现确定性 | ⭐⭐⭐⭐⭐ | 提分 = 付费，因果链清晰 |
| 监管风险 | ⭐⭐⭐⭐ | 标化考试不在双减范围 |

**结论：高度可行，与教育主线完美协同。**

## 二、执行指南

### 2.1 产品架构

```
自适应备考系统
│
├── 知识图谱引擎
│   ├── 各考试知识点拓扑图
│   ├── 题目-知识点映射
│   └── 用户能力模型（IRT / BKT 算法）
│
├── 自适应刷题 Agent
│   ├── 根据能力模型推送最优难度题目
│   ├── 错题自动归因（哪个知识点薄弱）
│   ├── 解析生成（AI 生成个性化解题思路）
│   └── 知识点掌握度可视化
│
├── 写作批改 Agent
│   ├── 托福/雅思/SAT 写作评分（对标官方标准）
│   ├── 逐句批注（语法/逻辑/用词/结构）
│   ├── 范文对比
│   └── 改进建议 + 高分句型推荐
│
├── 口语练习 Agent（托福/雅思）
│   ├── 语音输入 + 实时转写
│   ├── 发音评分
│   ├── 流利度/词汇/语法/逻辑评分
│   ├── AI 对话模拟（雅思 Part 2/3）
│   └── 录音回放 + 改进建议
│
├── 备考规划 Agent
│   ├── 目标分数 → 时间倒推
│   ├── 每日学习计划
│   ├── 薄弱环节优先策略
│   └── 动态调整（根据练习数据）
│
└── 模考系统
    ├── 全真模拟（计时 + 评分）
    ├── 分数预测
    ├── 历次模考趋势图
    └── 薄弱点分析报告
```

### 2.2 分阶段开发计划

```
Phase 1: 托福写作 + 自适应阅读（2026 Q2, 6 周）
├── 托福写作批改 Agent（市场验证最快的切入点）
├── 托福阅读自适应刷题（200+ 题目）
├── 基础知识图谱
└── 微信小程序

Phase 2: 全科覆盖（2026 Q3, 8 周）
├── 扩展到听力/口语
├── 增加雅思/SAT
├── 模考系统
├── 备考计划生成器
└── 语音交互（口语练习）

Phase 3: 深度个性化（2026 Q4）
├── IRT 自适应算法优化
├── 学习行为分析（何时学效率最高）
├── 社区（备考群/打卡/排行）
├── B 端（培训机构 OEM）
└── 接入 P0 教育辅导和 P1 留学规划
```

### 2.3 题库建设

| 数据来源 | 内容 | 合法性 |
|---------|------|--------|
| ETS/College Board 官方发布 | 真题 + 官方解析 | 合法（公开发布） |
| 公开题库整理 | TPO/OG/真题回忆 | 灰色（需法务审核） |
| AI 生成原创题 | 模拟真题风格 | 完全合法 |
| 用户贡献 | 考后回忆 | 灰色（可标注来源） |

**策略**：Phase 1 用公开真题 + AI 原创题，Phase 2 逐步建立大规模原创题库。

### 2.4 获客策略

| 渠道 | 方式 | 预估 CAC |
|------|------|---------|
| P0/P1 交叉推荐 | 现有教育/留学用户 | ~0 |
| 「AI 批改作文」免费体验 | 小红书/知乎传播 | 20-40 元 |
| 备考社群 | 托福/雅思打卡群 | 10-30 元 |
| 与培训机构合作 | 学员试用 | CPS |
| 考前冲刺广告 | 考试前 1 个月精准投放 | 50-100 元 |

**核心钩子**：「上传一篇托福作文，AI 30 秒给你官方评分 + 逐句批注」

## 三、商业模型方案

### 3.1 收入模型

| 套餐 | 月费 | 包含 | 目标用户 |
|------|------|------|---------|
| 免费 | 0 元 | 每天 3 题 + 1 篇作文批改 | 引流 |
| 单科版 | 69 元/月 | 单科无限刷题 + 批改 | 单项提分 |
| 全科版 | 149 元/月 | 全科 + 模考 + 备考计划 | 主力用户 |
| 冲刺版 | 299 元/月 | 全科 + 口语练习 + 每日计划 + AI 答疑 | 考前冲刺 |
| 考试包 | 799 元 | 全功能 3 个月（一个备考周期） | 一次性购买 |

#### B 端

| 模式 | 价格 |
|------|------|
| 培训机构 OEM | 30-80 元/学生/月 |
| 学校批量授权 | 2-5 万/年/校 |

### 3.2 单位经济模型

```
- ARPU = 149 元/月（全科版为主）
- 平均使用月数 = 4 个月（一个备考周期）
- LTV = 149 × 4 = 596 元（或 799 元考试包）
- CAC = 40 元
- AI 成本/用户/月 = 20 元
- 毛利率 = (149 - 20) / 149 = 87%
- LTV/CAC = 596 / 40 = 14.9x ✅
```

### 3.3 财务预测

| 指标 | 2026 Q3 | 2026 Q4 | 2027 Q2 |
|------|---------|---------|---------|
| 付费用户 | 500 | 2,000 | 5,000 |
| MRR | 74,500 | 298,000 | 745,000 |
| 月成本 | 30,000 | 80,000 | 180,000 |
| 月利润 | +44,500 | +218,000 | +565,000 |

### 3.4 风险与对策

| 风险 | 对策 |
|------|------|
| 题库版权争议 | 以 AI 原创题为主，真题仅做参考 |
| 评分不够准 | 用人工标注数据校准，对标官方评分 |
| 考试改革（如 SAT 数字化） | 快速适配新题型，反而是新入口 |
| 竞品多（赛道拥挤） | 深度个性化 + 与辅导/留学产品联动 |

## 四、与 P0/P1 的协同矩阵

```
完整用户旅程：

G5-G9 学科辅导（P0）
  ↓ 自然转化
G10 开始标化备考（本产品）
  ↓ 同期
G10-G11 活动规划（P1 留学）
  ↓
G11-G12 选校 + 文书（P1 留学）

→ 单用户全旅程 LTV: 792 + 596 + 1,995 = 3,383 元
→ 家庭版（2个孩子）: ~6,000 元
```

**共享**：用户系统、Mem0 记忆、技术栈、获客渠道。
