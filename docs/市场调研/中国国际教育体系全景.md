- 部分学校增设高考轨（双轨并行）
- 富裕家庭向日本/新加坡/泰国分流

## 2. 主流课程体系对比

| 维度 | IB (IBDP) | A-Level | AP | 加拿大 BC/OSSD |
|------|-----------|---------|----|----|
| **评估方式** | 内评+外评；6 科 + TOK/EE/CAS | 全外部考试 | 全外部考试 | 平时成绩 + 省考 |
| **计分** | 1-7 / 满分 45 | A*-E | 1-5 | 百分制 |
| **选科** | 6 组必选（3HL+3SL）+ 核心 | 3-4 科，深度专攻 | 灵活，4-8+ 科 | 每年 7 门学分课 |
| **广度 vs 深度** | 最均衡 | 最深 | 灵活 | 较均衡 |
| **主攻方向** | 全球通用，欧/加/美/英均认 | 英国/港新为主 | 美国为主 | 加拿大/澳洲 |
| **中国学校数** | 164 (DP) | 头部 60% 提供 | 头部 63% 提供 | 20+ BC / 若干 OSSD |
| **代表学校** | 包玉刚、UWC、德威、世外 | 深国交、领科、光华剑桥 | BASIS、北师大实验、人大附 ICC | 枫叶教育、中加枫华 |

### 头部学校课程分布

- 74% 提供 IB 课程
- 63% 提供 AP 课程
- 60% 提供 IGCSE/A-Level
- 55% 同时提供 2-3 种课程体系

- 2003 年创办，深圳首所全日制国际高中

## 7. 加拿大课程学校

### 枫叶教育 (Maple Leaf)

- 1995 年大连创办，中国最大加拿大课程集团
- **19 个城市**，约 33,000 学生
- 双文凭: 中国高中文凭 + BC 省高中文凭
- 上市教育集团

### 中加枫华 — 苏州

- 2003 年创办，BC 省教育部 + 江苏省教育厅双认证
- 3,000+ 学生 (3-18 岁)
- G7-G9: 数学/社会/科学/语言用英文教学
- G10-G12: 全 BC 课程 (每年 7 门学分课)

### OSSD 课程

- Bond Schools、Xspace Learning、树人瑞贝等提供安大略省课程
- 认可度主要在加拿大/澳洲方向

## 8. 其他国际课程

### 澳大利亚 VCE/WACE

| 体系 | 中国学校数 | 特点 |
|------|:---------:|------|
| VCE (维州) | 21 所 | 江苏最密集 (5 所)；自 2002 年起 7,217 名中国学生通过，94% 入读澳洲大学 |
| WACE (西澳) | 数所 | 北京爱迪首批认证 (2004) |

### 德国 DSD

- 2004 年进入中国，目前 **43 所初高中** 开设 DSD 课程
- 代表: 成都外国语学校、上海德国学校（两校区，140 所海外德国学校中最大之一）

### 法国课程

- **上海法国学校 (LFS)**: 1995 年创办，1,530+ 学生 / 60+ 国籍，学费 11.1-19.2 万/年
- **北京法国国际学校**: 1965 年创办
- 均隶属 AEFE (法国海外教育署)

### 日本 EJU

- EJU 考试**不在中国大陆设考点**（需赴日本或香港）
- 2025 年 14,044 名中国学生在日本参加 EJU
- 中国有大量日语预备校，但正式考试需出境

## 10. 政策环境

### 10.1 关键法规

**《民办教育促进法实施条例》(2021.9.1 生效)**

| 条款 | 影响 |
|------|------|
| 外资禁入义教 | 外资企业/外控社会组织不得举办、参与或控制 G1-G9 民办学校 |
| 外教材禁令 | G1-G9 民办学校禁用境外教材 |
| 公办接管禁令 | 民营企业不得收购公办学校 |
| 入学考试限制 | 禁止学科性入学考试 |
| 利润限制 | 义教阶段民办学校不得盈利 |
| IPO 禁令 | 民办学校不得上市融资 |

**双减政策 (2021.7.24)**

- 禁止 K-G9 学科类营利性培训（英语/数学/语文）
- 减轻课后辅导和作业负担
- 爱国教育法 (2024.1.1): 国际学校须融入中国课程要素

### 10.2 实际影响

| 影响面 | 说明 |
|--------|------|
| 外籍人员子女学校 | **不受影响**，可继续使用境外课程 |
| 双语学校 G1-G9 | 必须使用中方教材，融合国际元素 |
| 高中 G10-G12 | 可继续开设 IB/A-Level/AP |
| 市场规模 | 学校总数降至 972 所，在校生降至 ~49.6 万 |
| 招生冲击 | 幼小段冲击最大，高中段需求仍强 |
| 学校策略 | 增设高考轨、更名去"国际"、增加中方课程比例 |

### 10.3 对 BasisPilot 产品的启示

| 机会 | 说明 |
|------|------|
| **G10-G12 国际课程不受限** | AP/IB/A-Level 高中阶段需求稳健，测评系统可安全拓展 |
| **双语学校增长** | BASIS 双语校区从 1 所增到 5 所（含规划），需要中英双语测评 |
| **多课程体系需求** | 55% 学校提供 2-3 种课程，需灵活适配 IB/A-Level 题型 |
| **Tier 2 城市扩张** | 成都/武汉/嘉兴等新市场，测评系统可随校区落地 |
| **MAP 替代/补充** | 双减后学科培训受限，校内诊断性测评需求上升 |

## 11. 升学方向速查

```
升学美国:  BASIS(AP) > 公立国际部(AP) > IB 学校
升学英国:  深国交(A-Level) > 领科/光华(A-Level) > IB
升学加拿大: 枫叶/枫华(BC/OSSD) > IB > AP
升学澳洲:  VCE/WACE 学校 > IB > 枫叶
升学全球:  IB (最通用) > A-Level > AP
升学港新:  A-Level > IB > AP
升学日本:  日语预备校 + EJU (需出境考试)
性价比:    公立国际部 (8-12万) > 枫叶 (8-15万) > 民办双语 > 纯外籍学校
```

## 12. 题库扩展建议

基于以上体系分析，BasisPilot 题库可按优先级扩展：

| 优先级 | 课程体系 | 题型特点 | 预估扩展量 |
|:------:|----------|----------|:---------:|
| P0 (已完成) | AP/MAP (BASIS) | MCQ + fill-in，双语 | 352 题 ✅ |
| P1 | A-Level/IGCSE | 结构化简答 + MCQ，CIE 真题风格 | 500+ 题 |
| P2 | IB MYP | 开放题 + 评估标准 rubric | 200+ 题 |
| P3 | 加拿大 BC/OSSD | 学分课作业 + 省考 | 200+ 题 |
| P4 | SAT/ACT | 标准化测试 MCQ | 300+ 题 |
| P5 | IB DP | 论述题 + 内部评估模板 | 100+ 题 |

### A-Level 扩展重点

- 数学: Pure Mathematics + Statistics + Mechanics
- 物理: CIE 9702 真题风格
- 化学: CIE 9701 真题风格
- 英语: IGCSE English Language + Literature

### IB MYP 扩展重点

- 数学: Criterion-based assessment (A/B/C/D)
- 科学: Investigation design + data analysis
- 语言: Text types + literary analysis rubrics
